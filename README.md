## 📝 Descripción
Este repositorio centraliza todas las prácticas y ejercicios realizados en el módulo de **Programación**. Aquí se refleja mi evolución desde los conceptos más básicos hasta la arquitectura de software avanzada.

---

## 📂 Organización del Repositorio

| Sección | Módulos | Temática Principal |
| :--- | :--- | :--- |
| 🛠️ **Configuración** | `entornosPractica` | Setup del entorno y pruebas de librerías. |
| 🔢 **Fundamentos** | `practica1` - `practica4` | Sintaxis, tipos de datos y control de flujo. |
| 🏗️ **Estructuras** | `practica5` - `practica8` | Manejo de arrays, métodos y colecciones. |
| 💎 **Avanzado** | `practicaPOO` | Programación Orientada a Objetos. |

---

## 🚀 Herramientas Destacadas

* **Script de Generación:** `generar_proyecto_xrpg` (Utilidad para agilizar el flujo de trabajo).
* **Recursos:** La carpeta `/images` contiene los diagramas y esquemas del proyecto.

---

## ⚙️ Requisitos de Ejecución
1.  Tener instalado el **JDK 17+**.
2.  Importar el proyecto como un proyecto de **IntelliJ** (archivo `.iml` incluido).
